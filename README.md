# Block-Game
this is a multiplayer game in which players has make sure that the ball doesn't passes the brick

*[Game code](https://github.com/asenarmour/block-game/blob/master/test.java)

-----------

## Controls:

### Player 1:

A-left

D-right

W-up

S-down

### Player 2:

left arrow(<-) - left

right arrow(->) - right

up arrow - up

down arrow - down
